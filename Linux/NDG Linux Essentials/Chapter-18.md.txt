-Setuid
When the setuid permission is set on an executable binary file (a program) the binary file is run as the 
owner of the file, not as the user who executed it. This permission is set on a handful of system utilities 
so that they can be run by normal users, but executed with the permissions of root, providing access to 
system files that the normal user doesn't normally have access to.
