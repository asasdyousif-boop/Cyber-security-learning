Linux Essentials – Chapter 7Navigating the FilesystemIn this chapter, I learned how to move around the Linux filesystem and understand how directories are organized.I learned that the Linux filesystem starts from the root directory, and all files and folders are stored under it.I also learned how to find my current location, view the contents of directories, and move between different folders.Another important topic was understanding the difference between an absolute path, which starts from the root directory, and a relative path, which starts from the current location.I learned about special symbols used while navigating the filesystem, such as the current directory, the parent directory, the home directory, and the root directory.What I Learned- Understanding the Linux filesystem structure.- Navigating between directories.- Viewing files and folders.- Understanding absolute and relative paths.- Learning the purpose of common navigation symbols.ConclusionThis chapter helped me understand how to navigate the Linux filesystem more confidently. These concepts are important because they make it easier to work with files and directories in Linux.Commands Covered:pwdcd ..ls /var (used to list the contents of any directory in the filesystem.
