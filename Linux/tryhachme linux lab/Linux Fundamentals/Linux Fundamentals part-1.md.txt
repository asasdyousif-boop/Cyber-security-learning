Completed the first lab and successfully learned and applied the following commands.

echo : Output any text that we provide 
Ex: 
echo Hello 
output: 
Hello

Whoami : Find out what user were are currently logged in as!
Ex: 
whoami
output :
tryhackme 
 
ls : listing 
Ex :
ls 
output :
Show directories

cd : change ditectory
Ex : 
cd Desktop
 output :
It will enter the desktop.

pwd : print working directory
Ex :
pwd 
output : 
/home/ubuntu/Desktop

cat :   concatenate/output the contents of  file 
Ex: 
cat file.txt
output :
show file content

find : to find out what is where we can use commands such as find  
Ex:
find -name passwords.txt
output :
./folder1/passwords.txt

grep : The grep command allows us to search the contents of files for specific values that we are looking for.


& : This operator allows you to run commands in the background of your terminal.

&& : This operator allows you to combine multiple commands together in one line of your terminal.

> : This operator is a redirector - meaning that we can take the output from a command (such as using cat to output a file) and direct it elsewhere.
Ex:
echo hey > welcome
output:  Typing hey in the welcome directory

>> : This operator does the same function of the > operator but appends the output rather than replacing (meaning nothing is overwritten).
Ex : 
 echo hello >> welcome
output : Add hello to the welcome directory
